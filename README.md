# 🎮 3D-2048 Webgame

A modern 3D reinterpretation of the classic 2048 puzzle game.  
This game transforms the traditional 2D grid into a **4×4×4 cube**, allowing players to rotate perspectives and merge tiles in three-dimensional space. The experience is more strategic, more challenging, and much more fun.

Play Online:  
👉 **https://hzagaming.github.io/3D-2048-webgame/**
