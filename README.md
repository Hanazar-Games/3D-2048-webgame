# 🎮 3D-2048 Webgame

A modern 3D reinterpretation of the classic 2048 puzzle game.  
This game transforms the traditional 2D grid into a **4×4×4 cube**, allowing players to rotate perspectives and merge tiles in three-dimensional space. The experience is more strategic, more challenging, and much more fun.

Play Online:  
👉 **https://hzagaming.github.io/3D-2048-webgame/**

EN | English

3D-2048 Webgame is a modern reinterpretation of the classic 2048 puzzle, rebuilt inside a 4×4×4 three-dimensional cube.
Players can freely change perspectives, explore different viewing angles, and strategically slide blocks within 3D space.
Merging tiles becomes more dynamic and challenging compared to the traditional 2D version, offering deeper strategy and a refreshing gameplay experience.

This project is lightweight, fast, and fully playable in any modern browser.

ZH | 中文介绍

3D-2048 网页游戏是一款对经典 2048 的现代化重制版本，将传统 2D 网格升级为 4×4×4 的三维立方体。
玩家可以自由切换视角，从不同角度观察棋盘，并在立体空间中滑动方块进行合并。
相对于平面的 2048，本作在空间规划、策略深度和操作体验上都有质的提升，更具挑战性，也更具趣味性。

本项目无需安装，完全在浏览器中运行，轻量流畅。
